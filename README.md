# Zomato Restaurant Data Analysis

## Project Overview
This project analyzes Zomato restaurant data to understand how customer experience, pricing, and service features influence restaurant ratings.

The goal is to identify:
- Impact of online ordering
- Impact of table booking
- Best performing restaurant types
- Relationship between cost and customer satisfaction

## Dataset
The dataset contains 148 restaurants with:
- Restaurant name
- Online order availability
- Table booking
- Ratings
- Votes
- Cost for two people
- Restaurant type

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Insights
- Restaurants with online ordering have higher ratings
- Table booking improves customer satisfaction
- Buffet and casual dining perform best
- More votes lead to better ratings
- High price does not guarantee high ratings
